# GigwaWebsite

> Website [React JS](https://react.dev/) projects + [Vite](https://vitejs.dev/)

### Clone to local

```bash
git clone git@github.com:GuilhemSempere/GigwaWebsite.git
cd GigwaWebsite
npm install
```

## Usage

### Development

```bash
npm run dev
```

### Build & deploy on Github Pages

```bash
npm run deploy
```
